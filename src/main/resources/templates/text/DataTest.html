如何获取准确时间

  • 使用 new Date() 创建日期对象，它表示当前的日期和时间
  • 使用 Date.now() 或 new Date().getTime() 获取当前时间戳（从1970年1月1日至今的毫秒数）
  • 时间戳可以转换为各种时间格式：
    • getHours(), getMinutes(), getSeconds() - 获取时、分、秒
    • getFullYear(), getMonth(), getDate() - 获取年、月、日
    • toLocaleString() - 获取本地化格式的时间字符串
  • 使用 setInterval() 函数可以实时更新时间显示
